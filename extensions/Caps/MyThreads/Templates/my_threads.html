This page is used to manage all of your message threads with experts. Threads will show up in this list if one of the following conditions is true.

  • You are the creator of the thread.
  • You are an expert that has been matched up with a member.

If you would like to request for an expert consultation, click "Add Thread" and complete the fields. If you have been matched up with an expert, the "Matched?" column should change to "Yes".
